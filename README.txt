README

Included are the files CMPT_417_Assignment.cpp and qcp2cnf executable

To compile:
1 - Go to directory with CMPT_417_Assignment.cpp in command line
2 - type command g++ -o qcp2cnf CMPT_417_Assignment.cpp
3 - This will get you an executable with name qcp2cnf

To use program:
1 - enter in ./qcp2cnf <input> <output> in command line to create a cnf file
2 - use glucose-syrup SAT solver on above cnf file
